# DocStation Linux v8

## Изменения

- Нативное GTK-приложение запускается полноэкранно. Для отладки можно запустить оконно:
  ```bash
  DOCSTATION_WINDOWED=1 ./scripts/run_app.sh
  ```
- Задержка перед переводом следующего регистратора в режим диска уменьшена до `5000 мс`.
- На главном экране добавлена сетка ячеек док-станции: 5 колонок, до 20 стаканов.
- Ячейки показывают состояния: пусто, обнаружен, очередь подключения, пауза, переключение, подключён, скачивание, готово, ошибка.
- Статусы ячеек обновляются во время цикла импорта через SQLite status keys.
- В режиме ожидания приложение показывает регистраторы, которые видит в Ambarella/vendor mode, ещё до запуска импорта.
- В админском интерфейсе добавлена вкладка `Настройки`:
  - папка архива, куда сохраняются скачанные файлы;
  - задержка переключения в диск;
  - количество параллельных скачиваний.
- По умолчанию архив сохраняется в `/mnt/docstation_archive`.

## Важное про DockSlots.json

Файл `DockSlots.json` содержит калибровку физических стаканов. При обновлении рабочей станции не затирай его. Этот ZIP не должен перезаписывать уже откалиброванный `DockSlots.json`.
