# DocStation Linux v9 — карточки ячеек как в Windows-версии

В v9 главная сетка ячеек доведена ближе к Windows-интерфейсу:

- номер ячейки;
- состояние;
- камера / Device ID;
- сотрудник / Officer ID;
- индикатор `регистратор / зарядка`;
- строка памяти регистратора;
- прогресс памяти;
- общий прогресс скачивания;
- прогресс текущего файла;
- текущий файл / скорость / ETA;
- цветовые состояния: пусто, обнаружен, очередь, переключение, скачивание, готово, ошибка.

Ограничение: реальная батарея пока не читается из протокола. Как и в Windows-версии, показывается placeholder `регистратор / зарядка` для подключенного регистратора.
