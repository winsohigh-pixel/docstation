# Калибровка стаканов DocStation Linux v2

Калибровка связывает физическую ячейку док-станции с USB topology path Linux.

Используется не serial регистратора и не буква диска. У этих регистраторов serial часто одинаковый, а букв дисков в Linux нет. Надёжная привязка — физический путь через USB-хабы, например `3.1`, `3.7.4`.

## Запуск

```bash
cd ~/Downloads/docstation_linux
./scripts/calibrate_slots.sh 20
```

Можно указать другое число ячеек:

```bash
./scripts/calibrate_slots.sh 10
```

## Порядок

1. Вынуть все регистраторы.
2. Запустить калибровку.
3. Для каждой ячейки вставлять один регистратор только в указанную ячейку.
4. Дождаться записи `Ячейка XX -> usb_path=...`.
5. Вынуть регистратор и перейти к следующей ячейке.

Результат сохраняется в `DockSlots.json`.

## Проверка карты

```bash
./scripts/show_slots.sh
./scripts/list_amba.sh
```

После калибровки `list_amba.sh` будет показывать ячейку рядом с найденным регистратором.

## Формат DockSlots.json

```json
{
  "slots": [
    { "slot": 1, "label": "Ячейка 01", "usb_path": "3.1" },
    { "slot": 2, "label": "Ячейка 02", "usb_path": "3.2" }
  ]
}
```
