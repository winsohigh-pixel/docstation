# DocStation Linux v7 — автоматическая калибровка стаканов

Калибровка больше не требует нажимать Enter на каждом стакане.

Режим работы:

1. Вынуть все регистраторы.
2. Запустить калибровку.
3. Программа ждёт пустую док-станцию.
4. Программа просит вставить регистратор в ячейку 01.
5. Как только видит один стабильный Ambarella-регистратор, автоматически фиксирует `usb_path`.
6. Программа просит вынуть регистратор.
7. После исчезновения регистратора автоматически переходит к следующей ячейке.
8. Повторяется до последней ячейки.

Файл карты:

```text
DockSlots.json
```

Запуск:

```bash
./scripts/calibrate_slots.sh 20
```

Старый ручной режим с Enter оставлен только как аварийный:

```bash
sudo -E python3 -m docstation.main calibrate-slots --count 20 --manual
```

Во время автоматической калибровки одновременно должен быть вставлен только один регистратор — в текущую ячейку.
